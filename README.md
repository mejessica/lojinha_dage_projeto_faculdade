
# Lojinha Dage - Shop & Landing Page

Bem-vindo ao repositório do projeto **Lojinha Dage**, uma interface desenvolvida como trabalho acadêmico para as disciplinas de HTML5 e CSS3. O objetivo deste projeto é unir a funcionalidade de uma vitrine virtual de materiais de design com a estética minimalista e elegante que define a marca.

## 🖋️ Sobre o Projeto

A lojinha Dage foi concebida para ser mais do que uma simples loja; ela opera como uma **Landing Page** estratégica. A proposta visual foca no conforto do usuário, utilizando espaços em branco e uma tipografia bem selecionada para garantir que o portfólio de produtos — que inclui materiais autorais desenvolvidos por mim — seja o protagonista.

### Diferenciais da Interface:
* **Minimalismo Funcional:** Design limpo que facilita a navegação e a descoberta de produtos.
* **Experiência do Usuário (UX):** Foco em simplicidade e elegância, evitando poluição visual.
* **Essência da Marca:** Uso de elementos visuais que transmitem profissionalismo e criatividade.
* **Seção 'Sobre':** Integrada à página principal para contar a história e a visão do estúdio de forma orgânica.

---

## 🛠️ Tecnologias Utilizadas

Para a construção deste projeto, foram utilizadas tecnologias fundamentais de desenvolvimento web, focando em semântica e performance:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização avançada, incluindo o uso de:
    * Flexbox/Grid para layouts responsivos.
    * Custom Properties (Variáveis CSS) para manutenção de cores e fontes.
    * Transições suaves para uma experiência mais "confortável".
    * Estrutura HTML semântica e organizada;
    * CSS comentado;
    * Responsividade para pelo menos um dispositivo diferente;
    * Estilo visual coerente (cores, tipografia, espaçamento);

## 🎨 Design System

A paleta de cores e a tipografia foram escolhidas para refletir um ambiente de design profissional. O foco está no contraste equilibrado, permitindo que as descrições minimalistas dos produtos tenham leitura fluida e rápida.
